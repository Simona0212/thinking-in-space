## 📋 CLAUDE.md vs 实际实现对比

### 错误1：Qwen模型类选择错误 ⚠️⚠️⚠️

#### CLAUDE.md中的示例（行18-23）：
```python
from transformers import AutoProcessor, AutoModelForImageTextToText
processor = AutoProcessor.from_pretrained("<Model-Name>")
model = AutoModelForImageTextToText.from_pretrained("<Model-Name>")
```

#### 问题分析：
1. **`AutoModelForImageTextToText`在transformers中不存在！**
   - 这个类名在transformers库中从未存在过
   - CLAUDE.md中的示例代码本身就是**错误的**或**过时的**

2. **我的初始实现**：
   - 尝试使用`Qwen2VLForConditionalGeneration` ❌
   - 这导致了架构不匹配错误（Qwen3 ≠ Qwen2）

3. **正确的实现**：
   ```python
   from transformers import AutoModel
   model = AutoModel.from_pretrained(
       model_name,
       trust_remote_code=True  # 关键！
   )
   ```

#### ✅ 已修复：
- 现在使用`AutoModel + trust_remote_code=True`
- 移除了错误的dtype参数
- 让模型自动选择合适的dtype

### 错误2：LLaVA dtype参数错误 ⚠️⚠️

#### CLAUDE.md规范（行28, 181, 189）：
```python
model = AutoModel.from_pretrained(
    "lmms-lab/LLaVA-OneVision-1.5-8B-Instruct",
    trust_remote_code=True,
    dtype="auto"  # ← 明确指定！
)
```

#### 我的错误实现：
```python
# ❌ 错误：使用torch.bfloat16
model = AutoModel.from_pretrained(
    model_name,
    trust_remote_code=True,
    dtype=torch.bfloat16,  # 错误！应该是 "auto"
    ...
)
```

#### 为什么这是错误的？
1. CLAUDE.md **明确**指定`dtype="auto"`
2. 我却自作主张使用`torch.bfloat16`
3. 这导致了：
   - DeprecationWarning（torch_dtype过时）
   - 可能的精度/兼容性问题
   - 不遵守官方推荐配置

#### ✅ 已修复：
```python
model = AutoModel.from_pretrained(
    model_name,
    trust_remote_code=True,
    dtype="auto",  # ✅ 严格按照CLAUDE.md
    device_map=self.device,
    low_cpu_mem_usage=True
)
```

#### 我的问题：

2. 尝试导入不存在的类（`Qwen2VLForConditionalGeneration`）

def load_model(self):
    from transformers import AutoProcessor, AutoModel

    self.processor = AutoProcessor.from_pretrained(

### LLaVAOneVisionEvaluator
```python
def load_model(self):
    from transformers import AutoModel, AutoProcessor

    try:
        self.processor = AutoProcessor.from_pretrained(
            self.model_name,
            trust_remote_code=True
        )
    except:
        self.processor = None

    self.model = AutoModel.from_pretrained(
        self.model_name,
        trust_remote_code=True,
        dtype="auto",  # ✅ 严格遵守CLAUDE.md
        device_map=self.device,
        low_cpu_mem_usage=True
    )
    self.model.eval()
```

**关键改进**：
- ✅ 使用`dtype="auto"`（CLAUDE.md规范）
- ✅ 移除了不必要的类尝试
- ✅ 移除了torch_dtype fallback
- ✅ 简化了代码逻辑
